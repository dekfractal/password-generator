# password-generator
ランダムなパスワードを生成します。
## 使い方
8文字のランダムなパスワードを生成するには、次のようにします。[参考](https://github.com/dekfractal/password-generator/blob/master/test/index.html)
```JavaScript
let pg = new PasswordGenerator();
let password = pg.password(8);
```
